Ask a question about a GitHub repository using the DeepWiki CLI.

Usage: `/deepwiki-cli:ask <owner/repo> <question>`

Run the following command:

```bash
deepwiki-cli ask $ARGUMENTS
```

If the `deepwiki-cli` command is not found, inform the user to install it:

```bash
cargo install deepwiki-cli
```

or

```bash
brew tap hamsurang/deepwiki-cli
brew install deepwiki-cli
```
